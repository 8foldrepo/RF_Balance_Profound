::Setup instructions:::::::::::::::::::::::::::::::::::::::::::::::::
For users:
1. Download and unzip the RF_Balance_Profound distribution folder 
   from the releases page of the Github Repo (this is it)
2. Double click the setup_user_pc.bat script in the folder, and 
   reply 'y' to each step unless you already have the dependancy.
3. While running the script, do not close the terminal or reboot.
   Click through the installers using the reccomended settings,
   close each window when you are done, and return to the terminal.
4. After running the script YOU MUST REBOOT YOUR PC.
   The application will not work properly otherwise.
   If the application shows an error message try rebooting.
